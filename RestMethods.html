Class: RestMethods

RestMethods(client)

A class that includes some useful methods for interacting with Discord API in easier way, the client instance will have a property named rest that has access to this class that will be used throughout the library for easier access to the request methods wherever we have access to client instance. its not recommended to use this class its meant to be used by the library, but ok i will still document it for those who might be interested.
