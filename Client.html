Events

ANY

Emitted on any websocket event, useful if you want to customize the websocket messages and add extra events. though its deprecated use it only if you know what you are doing
Parameters:
Name Type Description
op Number The opcode of the event.
type String The event type.
data Object The data object from discord.
Deprecated:
  • Yes
Source:

Example
client.on("ANY", (op, type, data) => {
 if(op === 0 && type === "MESSAGE_CREATE") {
   console.log(data.content);
   // logs all message contents.
   // just an example don't be a spy.
  }
});
